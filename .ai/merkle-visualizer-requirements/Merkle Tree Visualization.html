How it works: Each leaf node hashes a data block. Parent nodes hash the concatenation of their children's hashes. The Merkle Root at the top uniquely represents all data. Changing even one byte in any block changes the root hash.

Try it: Add data blocks, hover over nodes to see full hashes, and click Verify Proof to see a Merkle proof path highlighted in gold.
